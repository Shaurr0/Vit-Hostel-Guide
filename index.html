Block Details

A block (MHA)

This is the oldest and first hostel block, built when vit was first established, has undergone many renovations to be upto date.Closer to academics block for mechanical and civil students.5 floors block

R Block (MHR)

R block is a newer hostel block built in 2019. It has apartment type rooms, has 5 different wings from A-E, each wing has 3 different rooms, and those 4 rooms get to share one washroom and one couch too.Overall on the expensive side.18 floors block

J Block (LHJ) [Girls Block]

The J block girls hostel is also a new entry, established in 2023. Entry is restricted after 9 PM.This one also offers Apartment type rooms, it is near to the main gate .
