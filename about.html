About

The person behind this project.

I'm Shaurya Pandey (24BCE0738). This project the VIT Hostel Guide was built as part of my web development coursework, and honestly it turned out to be one of the more rewarding things I've worked on this semester.

Before this course, I had a working understanding of programming, but web development always felt like a separate world. Going through this properly gave me a chance to actually get comfortable with HTML, CSS, JavaScript, and Bootstrap not just knowing they exist, but being able to use them with some confidence. Those are now skills I can genuinely put on my profile, and that matters.

Outside of coursework, I try to stay active. I participate in hackathons fairly regularly I find that working under time pressure on something that has to actually function is a different kind of learning. My GitHub reflects that; it's been consistently active with projects across different stages of completion.

For this project specifically, I used HTML, CSS, JavaScript, Bootstrap for the frontend, and hosted it on Vercel. No templates, no page builders everything was written by hand, which is something I wanted to be intentional about.

Also sir,If any of my work seems worth a closer look/potential , I would genuinely welcome the discussion. Please Feel free to reach out to me, I'd be glad to collaborate or take it further.
