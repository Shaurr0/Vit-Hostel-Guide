# VIT Hostel Guide

A student made guide to hostel blocks at VIT Vellore, built as part of my web development project. The site covers rough block wise information including capacity, warden contacts and block descriptions, intended as a practical reference for students, mainly freshers who find it difficult in the start.

Built using HTML, CSS, JavaScript and Bootstrap. 

Also hosted on Vercel.
